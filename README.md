# My 'Advent of Code 2021' Solutions

*[Advent of Code](https://adventofcode.com/)* is an Advent calendar of small programming puzzles for a variety of skill sets and skill levels that can be solved in any programming language you like.

## Dev Environment

Virtual environment: **aoc**, can be recreated using *requirements.txt*

```bash
$ mkvirtualenv aoc
$ pip install -r requirements.txt
```

You need to [set your session ID](https://github.com/wimglenn/advent-of-code-wim/issues/1) in either an environment variable or a file before you can download your personalized data with `advent-of-code-data`.
